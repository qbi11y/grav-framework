Getting Started

The Gravitant UI Framework was created to ensure there is a consistent user experience throughout all of Gravitant's products. Use this framework to copy and paste code snippets to speed up the development process.

In order to leverage the framework copy and paste the following code into the <head> tag of the page being developed.

<!--Font references-->
<link href='http://fonts.googleapis.com/css?family=Open+Sans:400,600' rel='stylesheet' type='text/css'>
<link id="gravitant-fonts-css" media="all" type="text/css" href="http://tools.gravitant.com/wp-content/themes/gravitant-evaluate/css/klinic-slab.css?ver=3.9.2" rel="stylesheet" />

<!--Icons-->
<link href="http://framework.gravitant.net/css/font-awesome.min.css" rel="stylesheet" />

<!--Gravitant Theme-->
<link href="http://framework.gravitant.net/css/gravitant-toolkit.css" rel="stylesheet" />

<!--Script Libraries-->
<script src="http://framework.gravitant.net/js/gravitant-toolkit.min.js"></script>

Page Templates

A great place to start the development is from a page template. The template has all of the high level items needed, simply add the data to template to bring it to life. There are three types of the page templates that can be leveraged. Expanded List Template, Table Template, and Details Template.

Theme Generator

The default theme that ships with cloudMatrix can be replaced with a new theme. In order to replace the default theme with a new theme, a new theme must first be generated*. To generate a new theme, simply download the theme generator and follow the following instructions:

  1. Unzip package
  2. Open gravitant-toolkit-alt.less
  3. Update the variables to reflect desired changes.
  4. Save gravitant-toolkit-alt.less
  5. Compile gravitant-toolkit-alt.less into theme.css
  6. Upload theme.css to ...

*Please note you will need a LESS compiler to generate a theme. You can get more information and download a free compiler at lesscss.org
