Gravitant UI Framework

Ensuring Gravitant's products are consistent throughout.

Learn more

The Gravitant UI Framework was created to ensure there is a consistent user experience throughout all of Gravitant's products. Use this framework to copy and paste code snippets to speed up the development process.

In order to leverage the framework please include the following code on your page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Gravitant UI Framework</title>
    <!--Site Icon-->
    <link rel="icon" href="assets/images/favicon.ico" type="image/x-icon">

    <!--Font references-->
    <link href='http://fonts.googleapis.com/css?family=Open+Sans:400,600' rel='stylesheet' type='text/css'>
    <link id="gravitant-fonts-css" media="all" type="text/css" href="http://tools.gravitant.com/wp-content/themes/gravitant-evaluate/css/klinic-slab.css?ver=3.9.2" rel="stylesheet">
    <link href="assets/fonts/frutiger.css" rel="stylesheet" />

    <!--Icons-->
    <link rel="stylesheet" href="assets/font-awesome.min.css">

    <!--Gravitant Theme-->
    <link href="assets/gravitant-toolkit.css" rel="stylesheet" />

    <!--Script Libraries-->
    <script src="js/gravitant-toolkit.min.js"></script>

</head>
<body id="gravitant-toolkit">
    <div class="page-wrapper">

        <header>
            <nav class="navbar navbar-default navbar-fixed-top">
                <div class="container">

                    <div class="navbar-header">
                        <button aria-controls="navbar" aria-expanded="false" data-target="#navbar" data-toggle="collapse" class="navbar-toggle collapsed" type="button">
                            <span class="sr-only">Toggle navigation</span>
                            <span class="icon-bar"></span>
                            <span class="icon-bar"></span>
                            <span class="icon-bar"></span>
                        </button>
                        <a class="" href="index.html">
                            <img src="assets/images/gravitant-inverse-logo.png" alt="logo" style="max-height: 55px;" />
                        </a>
                    </div>
                    <div class="collapse navbar-collapse" id="navbar">
                        <ul class="nav navbar-nav">
                            <li  class="active"><a href="#/" >Home <span class="sr-only">(home)</span></a></li>
                        </ul>
                    </div>
                </div>
            </nav>
        </header>
        <section class="container">
            <div></div>
        </section>
        <footer>
            <div class="container">
                <ul class="list-unstyled list-inline">
                    <li class="pull-right">
                        <img src="assets/images/powered_by_gravitant.png" alt="footer logo" />
                    </li>
                </ul>

        </footer>
    </div>

</body>
</html>
