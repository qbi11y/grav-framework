Components

Components are a collection of elements that represent a user interface task. Use multiple components to build out a page.

If you feel the provided components do not offer the support you are looking for, please send feedback so the current set of components can be addressed.

Page Action Bar

The page action bar contains page level actions that affect all items on the page. By default the page action bar allows a user to sort by, search, filter and perform actions as they relate to the content.

Customization

The page action bar can easily be customized using the theme generator. The background and outline colors can be customized.

Please note that the background and outline colors are the same for all the wells throughout the application. If the colors are updated in one location they will be updated in all of the locations.

Filter

The filter component should be used when a large dataset is returned and it would be reasonable that a user would want to filter out data to get exactly what they are looking for. The filter should be triggered in the page action bar component with the filter window being displayed below the page action bar.

Breadcrumb

Breadcrumbs are used to quickly navigate to a specific section. Bresdcrumbs are not required if a section is one level deep. If a section is two or more levels deep a breadcrumb is needed for all of the sections. All of the breadcrumb links should be links that link to respective section. The last breadcrumb should represent the current page and should not be a link.

Example

  1. Breadcrumb 1
  2. Breadcrumb 2

Alerts

Alerts are used to communicate with the end user. Utilize alerts when the system has returned an error, success or a warning. If there is information that will help the user within the product use the information alert.

Example

The service was successfully edited
The service is in progress
A service warning has occurred
A service error has occurred

Collapsible Sub Section

When content on a details page impacts how much content can be shown at once, consider using a collapsible sub section. This will allow a user to control what they see and when while allowing for the most content to be shown on the screen.
