Templates

Templates are a way to quickly get a page type up and running with minimal effort. Utilizing a template will ensure that Gravitant UX style guidelines are met with minimal development effort required

If you feel the provided templates do not offer the support you are looking for, please send feedback so the current set of templates can be addressed.

Expanded List Page

This template provides the framework for an expanded list page. It includes a header, footer, page action bar and layout for the list. Place your custom content in the space provided to ensure compliance with the Gravitant UX guidelines.

Usage

Use the expanded list page when large amounts of data need to be shown. If the data to be displayed would take up 7 or more columns in a table, strongly consider using the expanded list page. The actions exposed should be frequent actions and must not exceed three. All other actions must be placed in the 'More Actions' drop down.

Basic Table Page

This template provides the framework for a basic table. It includes a header, footer, page action bar and layout for the list. Place your custom content in the space provided to ensure compliance with the Gravitant UX guidelines.

Usage

Use the basic table page when small amounts of comparative data need to be shown. If the data to be displayed would take up 7 or more columns in a table, strongly consider using the expanded list page. A table with 7 or more columns makes it difficult for the user to consume the data displayed.

To handle excessive columns leverage the Quick View feature of the table. The Quick View feature will allow a user to see extra column data without having to view a details page.
